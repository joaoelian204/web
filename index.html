Proyecto 2

El proyecto es una plataforma de cine desarrollada en Python, utilizando la biblioteca Tkinter para la interfaz gráfica. La plataforma permite a los usuarios explorar y gestionar información relacionada con películas. Además, incorpora una base de datos para almacenar tanto los datos de los usuarios como la información detallada de las películas, proporcionando una experiencia de usuario integral y eficiente.
